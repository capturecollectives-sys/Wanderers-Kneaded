# Gallery Image Names

Upload final image files into this folder using these simple names:

- `t1.jpg`, `t2.jpg`, `t3.jpg` for trailers available to rent.
- `f1.jpg`, `f2.jpg`, `f3.jpg` for food and pizza close-ups.
- `p1.jpg`, `p2.jpg` for places, past events, and where the van has been.

The website already points to these filenames. If a file is missing, the gallery falls back to temporary existing Wanderers Kneaded images so the layout does not look broken.
